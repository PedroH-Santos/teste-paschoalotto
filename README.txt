Olá tudo bem? segue algumas instruções sobre o teste feito


1 - A api abrange todo o conteúdo do tópico 1,2,3 e 4 do teste técnico, na qual possui uma rota para exportação dos dados, autenticação e cadastro/listagem dos usuários cadastrados no PostgreSQL

2 - No projeto do Visual Basic é preciso criar uma pasta no repositório: C:\arquivo\output.csv. Para que seja possível salvar o arquivo de exportação dos dados

3 - O banco de dados está com o nome de: Teste-Paschoalotto. E Para rodar as migrations do Entity Framework ( Criar a estrutura do banco) é preciso utilizar o comando : Add-Migration PreparandoBanco e dotnet ef database update --project ApiTestePaschoalotto no gerenciador de pacotes.
